# photobookmemory

L'idea e' quella di avere un book che ripercorre gli anni della vita con i momenti piu' importanti.
Ci saranno 3 step
Data input -> Elaborazione dati -> output

### Data input
in questa fase pensavo di collegare tutti i social e i vari cloud dove potrebbero essere presenti delle foto
Esempio:

- instagram
- Facebook
- Drive
- Dropbox
- Icloud
- etc

### Elaborazione
in questa fase prendiamo tutte le foto e le ordiniamo su una linea temporale, successivamente si proporranno le foto con piu' reactions come foto principali e si predispone il book. 
Quindi l'output sono 2: 

- tutte le foto ordinate cronologicamente
- Le principali foto di ogni evento importante della vita

Potrebbe essere utile avere in input il numero di foto rilevanti che si voglio avere

### Output

Gli output potrebbero essere diversi: 

- mettere tutte le foto su cloud esterno (affidarsi a cloud esterno tipo google drive o Icloud)
- stampare il book e spedirlo a casa (affidarsi a servizi esterni)
- Salvare su un cloud nostro (funzione avanzata)
